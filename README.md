# svgtk-portal-analytics
Diploma project - SVGTK Portal Information System: Analytics &amp; Reporting module with dashboard (KPIs), academic performance, attendance and graduation reports. Built with PHP &amp; MySQL.

Модуль: Система аналитики и отчетности для портала SVGTK.

Технологии: PHP, MySQL, CSS.
Для запуска импортируйте svgtk_portal.sql и откройте проект на Open Server Panel.
